## 변경 본질

전 문단 저장 lineseg 부재(기계생성) 문서에서 텍스트-host tac 표의 높이가
제목 줄높이(17.3px)로 붕괴하던 2중 결함:

1. fmt 기반 높이 분기 보정 — `para.line_segs.is_empty()` && 선택 높이 <
   측정 높이(`ft.total_height`)면 측정 높이 채택
2. "TAC 표 높이 보정" cap 을 lineseg 없는 문단에서 스킵 — cap 의 두 축 모두
   lineseg/컴포즈 반영을 전제하므로 전제 불성립 케이스 격리

**저장 lineseg 보유 문서는 두 변경 모두 불변** — 조건이 저장 필드
(`line_segs.is_empty()`) 기반 케이스 구조 가드.
