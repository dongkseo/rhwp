## 정정 방침

이월 그룹을 "같은 문단의 **진짜 flow 스택 float**"로 한정. 선행 항목 필터에서
소스 컨트롤을 조회해 다음만 인정:

```
treat_as_char == false  &&  text_wrap == TopAndBottom  &&  vert_rel_to == Para
```

이는 같은 함수(typeset_block_table)에서 이미 쓰이는 `is_para_topbottom` 정의와 동일.
- `tac=true` 캡션 상자 제외 → 156767631·78842 해소.
- `vert=용지`(Paper/Page) 절대 앵커 제외 → 3143097 해소.
- 2448877(별표4) 표1(tac=false, TopAndBottom, Para)은 그대로 인정 → 정합 유지.
