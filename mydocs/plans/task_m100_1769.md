# Task #1769 수행계획서 — 밀집 셀 per-line 측정 누적 조사 (판별 종결)

## 배경
#1765 판별에서 17931383 row9(+5.2px)를 per-line 콘텐츠 측정 누적(+0.1px/줄)으로
재분류(이슈 #1769). 본 태스크에서 줄 단위 1:1 대조로 검증.

## 판별 (조사 결과 — 가설 기각)
- row9 c2 셀 = 실제 **136줄** 콘텐츠(23문단), 저장 lineseg 에 vpos 리셋 2회 —
  원 문서에서 행 내부 분할(intra-row split)로 3쪽에 걸치던 셀.
- rhwp p2 fragment 50줄의 advance 를 저장 lineseg 와 1:1 대조: **누적차 +0.0px, 정확 일치**
  → per-line 드리프트 없음 (#1759 서베이의 pitch median 0.00 과 정합 — 줄 메트릭 건강).
- 잔여 +5.2px = **행 내부 분할 컷(fragment) 경계의 미세 오프셋** (rhwp fragment 818.5px vs
  한글 813.5px). 해당 파일의 페이지·PI 판정은 MATCH — 페이지 무영향, 쪽 내 미세 시각 차.

## 처리
- 이슈 #1769 재분류 코멘트: per-line 기각 → "분할 컷 미세 오프셋" (우선순위 낮음,
  페이지 무영향) — #1759 프로그램 항목으로 병기. 소스 무변경 종결.
- 이로써 razor-thin 분해 체인 완료: 표 선언높이(#1763 수정) / 측정 노이즈(판별) /
  per-line(기각·건강 확인) / 분할 컷(저우선 기록).
